# Replication-Files-for-BBCM2025
This repository includes the analysis and supplementary data files and stata.do files that replicate Tables 1 - 3 in  "On terms of trade, offshoring ties, and the enforcement of trade agreements"  by Subhayu Bandyopadhyay, Arnab K. Basu, Nancy H. Chau, and Devashish Mitra Journal of Economic Behavior and Organization 238 (2025) 107215 (BBCM_2025)
