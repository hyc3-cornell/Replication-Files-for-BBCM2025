This folder includes the analysis and supplementary data files and stata.do files that replicate Tables 1 - 3 in 
"On terms of trade, offshoring ties, and the enforcement of trade agreements" 
by Subhayu Bandyopadhyay, Arnab K. Basu, Nancy H. Chau, and Devashish Mitra
Journal of Economic Behavior and Organization 238 (2025) 107215 (BBCM_2025)

"analysis_final0322.dta" is the main analysis data file. 

"highincome_dum.dta" is a supplementary data file.

Table_1.do replicates the summary statistics table (Table 1) -- outputs "tabsum1.tex"
Table_2.do replicates the summary statistics table (Table 2) -- outputs "tabttest2.tex"
Table_3.do replicates the regression table (Table 3) -- outputs "tab3.tex"
